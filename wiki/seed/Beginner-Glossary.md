# Beginner Glossary

## Audio endpoint

A device that Windows can treat as a speaker, headphone output, or microphone input.

## XInput

Microsoft's API for reading Xbox-style controller state and sending rumble commands.

## Rumble frame

One time slice of motor output, consisting of a large-motor value, a small-motor value, and a
duration.

## Amplitude envelope

A simplified description of how strong a signal is over time, without preserving all of its
fine-grained waveform detail.

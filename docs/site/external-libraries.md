# External Libraries

This page exists so that a beginner can quickly jump from this project into the primary
documentation of the libraries it uses.

## Runtime and application libraries

- [CLI11](https://github.com/CLIUtils/CLI11): command-line parsing
- [spdlog](https://github.com/gabime/spdlog): structured console and file logging

## Testing and build libraries

- [GoogleTest](https://google.github.io/googletest/): unit and integration-style tests
- [CMake](https://cmake.org/documentation/): project configuration and builds

## Platform APIs

- [XInput documentation](https://learn.microsoft.com/windows/win32/xinput/getting-started-with-xinput): controller access on Windows

## Documentation tooling

- [Doxygen](https://www.doxygen.nl/manual/index.html): API documentation
- [MkDocs](https://www.mkdocs.org/): static documentation site generation
- [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/): documentation theme
- [Gollum](https://github.com/gollum/gollum): serveable local wiki software
